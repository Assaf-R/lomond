### net_probe

### Makefile tricks

## To compile
```
make
```
## To load compiled LKM
```
make i
```
## To remove loaded LKM
```
make r
```
## To ping google.com once (easy networking check)
```
make t
```
## To compile, load compiled LKM, and ping google.com
```
make full
```
## To print kernel logs (dmsg) and filter with our keyword
```
make d
```
